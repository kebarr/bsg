Detailed Description

template<class RecordType, class RecordFactory, class FileReader, typename FileRecord, typename ReaderParamStruct, typename FactoryParamStruct>
class SMR< RecordType, RecordFactory, FileReader, FileRecord, ReaderParamStruct, FactoryParamStruct >

SMR is an external memory map reduce engine with configurable memory bounds it requires a RecordFactory and a FileReader. The FileReader takes elements from the file and sends them to the RecordFactory for parsing into single RecordType elements.

Template Parameters
RecordTypeThe RecordType class requires strict weak ordering and equivalence comparison functions.
RecordFactoryGenerates a vector of RecordType elements to be merged into a single set.
FileReaderGets a single FileRecord from the input file.
FileRecordFileRecord containing zero or more RecordType elements.
ReaderParamStructParameters for the FileReader constructor.
FactoryParamStructParameters for the RecordFactory constructor.

void SMR< RecordType, RecordFactory, FileReader, FileRecord, ReaderParamStruct, FactoryParamStruct >::mapElementsToBatches ( FileReader &  myFileReader,
uint64_t &  numReadsReduction 
)
inlineprivate

Loops over the records on myFileReader and generates a vector<RecordType> which is then appended to the _elements vector, when the size of that vector is >= to the limit, a batch is sent to disk. If the number of batches on the disk equals the mergeCount, a merge is done for the records in memory and the records on the disk. This structure creates mergeCount batches and then reduces to one batch after the merge.

Parameters
myFileReaderThis is an already initialised FileReader class.
numReadsReductionThe number of file records generated by myFileReader.
